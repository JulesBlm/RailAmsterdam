# RailAmsterdam
[View the map](https://julesblm.github.io/RailAmsterdam/)

## English
Map made using D3.js of all tram/metro/train lines and stops in Amsterdam. It's open-source under GNU General Public License v3.0.

## Nederlands
Kaart van alle tram/metro/trein lijnen en haltes/stations in Amsterdam en omstreken gemaakt met D3.js en open data van de gemeente Amsterdam en NS. Open source onder GNU General Public License v3.0
